## 概要
ウェブ・マジシャン(Webmagician)はWebクローラープラットフォームです。「Less Coding, More Power」という基本理念に基づいて作られている。<br/>
最大のアクティビティと展開性を取得するために、プログラムの脚注で開発をサポートします。
